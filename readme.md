How to run this project:

1. Navigate to the Server folder, build the server.go file, or copy-paste one of the following commands:
- Linux: go build server.go && clear && ./server
- Windows: 
  - go build server.go
  - ./server

2. In a separate terminal, open your client using "go run Client.go" inside the Client folder
3. In case the Exit command does not work in the Client, use Task Manager, "pkill server" on Linux, or turn your computer off and on again.
